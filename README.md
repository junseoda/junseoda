# junseoda

> Computer Science student building data-driven web services with recommendation systems, public data, and automation.

컴퓨터공학을 기반으로 데이터를 정리하고, 문제를 정의하고, 실제 동작하는 웹 서비스와 평가 가능한 추천 시스템으로 구현하는 개발자입니다.

## About Me

- 관심 분야: Web Service, Data-driven Product, Recommendation System, Public Data, AI/Automation
- 공공데이터와 실제 서비스 흐름을 연결해 추천, 평가, 대시보드 형태로 구현하는 작업에 관심이 있습니다.
- 기획, 데이터 전처리, 백엔드 API, 프론트엔드 화면, 평가 지표 설계까지 한 프로젝트 안에서 연결해 보는 것을 중요하게 생각합니다.
- 대표 프로젝트는 선거 유세 장소 추천 시스템이며, 실제 후보 공개 일정 기반 Gold Set과 공공데이터를 활용해 추천 결과를 검증했습니다.
